Creativity

Imagine all repeatable tasks are automated.

It follows that the only remaining way for us to create value is to build something entirely new for society.

But to do this, we need to have a unique set of knowledge and insights.

Only then can we make the uniquely-human creative leaps needed to create something from nothing.

(Note: there’s no point worrying if AI/robots can be creative like this, because if they can, jobs become irrelevant anyway. This book may ease your concerns.)

But that doesn’t answer the question: how do we develop this knowledge to set ourselves apart?

Good question. Here’s how.
